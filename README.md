# وب‌سایت آجر جهان‌نما

نسخه استاتیک، واکنش‌گرا و آماده انتشار روی GitHub Pages.

## نمایش روی کامپیوتر

فایل `index.html` را باز کنید یا با افزونه Live Server اجرا کنید.

## انتشار در GitHub Pages

1. یک Repository جدید در GitHub بسازید.
2. تمام فایل‌های این پوشه را در ریشه Repository آپلود کنید.
3. وارد **Settings → Pages** شوید.
4. در بخش **Build and deployment**، گزینه **Deploy from a branch** را انتخاب کنید.
5. Branch را روی `main` و Folder را روی `/(root)` بگذارید و **Save** را بزنید.
6. چند دقیقه بعد لینک سایت در همان صفحه نمایش داده می‌شود.

ساختار پروژه بدون نیاز به Node.js یا نصب بسته اجرا می‌شود. شماره تماس و لینک واتساپ روی `09132675275` تنظیم شده‌اند.
